# 📋 Contact Form with JavaScript Validation

A simple contact form built with HTML, CSS, and JavaScript that include form validation and display successful message.


## ✨ Features

- ✅ **Name validation** — Only allows letters, spaces, hyphens, and apostrophes
- ✅ **Password validation** — Requires uppercase, lowercase, number, and special character (8+ chars)
- ✅ **Message validation** — Requires 80+ characters with allowed punctuation
- ✅ **Real-time error messages** — Shows specific errors under each field
- ✅ **Success message** — Displays confirmation when form is submitted successfully
- ✅ **Auto form reset** — Clears all fields after successful submission
- ✅ **No page reload** — Uses `event.preventDefault()` for smooth UX
